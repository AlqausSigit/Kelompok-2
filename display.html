Demo: CSS Display Property

Properti display di CSS digunakan untuk menentukan bagaimana elemen HTML ditampilkan dalam layout. Di bawah ini adalah contoh penggunaan berbagai jenis display beserta penjelasannya.

1. Display: Block

Kegunaan: Elemen block menempati seluruh lebar kontainer dan selalu mulai dari baris baru.

Contoh elemen block: <div>, <p>, <h1> sampai <h6>

Block 3

2. Display: Inline

Kegunaan: Elemen inline hanya selebar kontennya dan tidak mulai dari baris baru. Tidak bisa mengatur width dan height.

Contoh elemen inline: <span>, <a>, <strong>, <em>

Inline 1 Inline 2 Inline 3

3. Display: Inline-block

Kegunaan: Seperti inline karena bisa sejajar horizontal, tetapi juga seperti block karena bisa diatur ukuran.

Inline-Block 1
Inline-Block 2
Inline-Block 3

4. Display: Flex

Kegunaan: Flexbox digunakan untuk membuat layout yang fleksibel dan responsif. Elemen anak bisa diatur arah (horizontal/vertikal), diratakan, diberi jarak otomatis, dll.

Cocok untuk: Navbar, layout kolom, dan penataan baris fleksibel.

5. Display: Grid

Kegunaan: CSS Grid Layout memungkinkan penataan elemen dalam baris dan kolom. Ideal untuk layout dua dimensi.

Cocok untuk: Galeri, dashboard, halaman majalah.
