Dasar Flexbox di CSS

Flexbox (Flexible Box Layout) adalah sistem layout satu dimensi yang memungkinkan penataan elemen secara fleksibel dan responsif baik secara horizontal (baris) maupun vertikal (kolom).

1. Flexbox Arah Baris (default)

Container menggunakan display: flex; dan setiap item akan ditata secara horizontal (sejajar ke samping). Properti flex: 1 membuat semua item membagi ruang secara merata.
