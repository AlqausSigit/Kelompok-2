1. Position: static (default)

Default posisi elemen. Elemen akan ditampilkan mengikuti alur normal dokumen (tanpa bisa digeser dengan top, left, dll).
STATIC

2. Position: relative

Tetap di tempat aslinya, tapi bisa digeser. Contoh: geser 30px ke kanan (left: 30px)
RELATIVE

3. Position: absolute

Posisi diatur relatif terhadap parent yang "relative". yang berwarna hitam disekitar kotak absolute yang berbentuk memanjang dinamakan dengan absolute container.
ABSOLUTE

4. Position: fixed

Selalu menempel di layar (pojok kanan bawah), walau discroll.
FIXED

5. Position: sticky

Awalnya biasa, tapi akan "menempel di atas" saat scroll sampai posisinya.
STICKY
